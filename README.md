#  AUST PIC - Guess or Regret
#Link : https://fungame-weld.vercel.app/
A cyberpunk-themed interactive web game built for the **AUST Programming and Informatics Club** fair stall. This app features a "Hacker" style UI to engage students with logic puzzles and games.

##  Game Modes

### 1.  Number Hack
* **Objective:** Guess the secret passcode (Number between 01-100).
* **Mechanics:** You get 5 attempts. The system tells you if the number is "Higher" or "Lower".
* **UI:** Battery life indicator for attempts, history log, and shake animations for wrong guesses.

### 2.  Brain Teaser
* **Objective:** Solve Computer Science, Math, and Logic riddles.
* **Mechanics:** You have **3 attempts** per question.
* **Content:** Questions range from basic logic (Binary, GCD) to programming trivia.
* **Hint System:** A hint appears if you are on your last attempt.

##  Tech Stack

* **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Effects:** [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti)

##  How to Run

1.  **Clone the repository** (or download files):
    ```bash
    git clone [https://github.com/e/guess-or-regret.git](https://github.com/your-username/guess-or-regret.git)
    cd guess-or-regret
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Run the App:**
    ```bash
    npm run dev
    ```

4.  **Open in Browser:**
    Go to `http://localhost:5173`
